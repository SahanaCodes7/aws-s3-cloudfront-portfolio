#  AWS S3 + CloudFront Portfolio Hosting

This project demonstrates how I hosted a **static portfolio website** using **Amazon S3** and distributed it securely with **Amazon CloudFront**.

---

##  Project Overview
- **Amazon S3** ‚Üí stores static website files  
- **Amazon CloudFront** ‚Üí CDN for secure global delivery (HTTPS)  
- **IAM & Bucket Policies** ‚Üí manage permissions  
- **AWS Budgets** ‚Üí cost monitoring to stay within Free Tier
